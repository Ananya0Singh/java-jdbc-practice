#  JDBC Practice

Welcome! 👋

This repository contains my Java JDBC practice programs along with a mini project developed while learning Java Database Connectivity (JDBC).
It helped me understand how Java applications communicate with relational databases and perform common database operations.

## 📚 Topics Covered

- JDBC Connection
- CRUD Operations
- Insert & Update Queries
- PreparedStatement
- Scrollable ResultSet
- Exception Handling
- Database Connectivity
- Practice Programs

## 🚗 Mini Project – Garage Billing System

The **Garage Billing System** is a console-based Java application built using JDBC.

The project is organized into different layers such as configuration, entities, and services to keep the code clean and modular.

It includes:

- 👤 Customer Management
- 🧾 Invoice Management
- 💰 Billing Operations
- 🔗 Database Connectivity using JDBC

This project helped me understand how database-driven Java applications are structured and how different components work together.
